🔌 Energy Consumption Predictor

Predict building energy usage with machine learning for smarter sustainability

📌 Project Overview

The Energy Consumption Predictor is a regression-based ML model that estimates building energy usage based on user inputs. Ideal for facility managers, energy analysts, and sustainability researchers looking to improve energy planning.

🧠 Features Used

Feature Description
Building TypeCategorical feature specifying the type of building (e.g., Residential, Commercial).
Square FootageTotal area of the building in square feet.
Number of OccupantsTotal number of people regularly occupying the building.
Appliances UsedCount of energy-consuming appliances installed.
Average TemperatureAverage outdoor temperature in °C.
Day of WeekCategorical value indicating the day (Monday to Sunday).
Energy ConsumptionTarget Variable – Measured in kWh (kilowatt-hours).

🛠️ Technologies Used

  • Python
  • Pandas / NumPy
  • Matplotlib / Seaborn
  • Scikit-learn (Linear Regression, StandardScaler)
  • Flask
  • HTML / CSS

🚀 Future Improvements

  • Incorporate time-series data (hourly/daily usage)
  • Use XGBoost or Random Forest for improved accuracy
  • Deploy to cloud (AWS, Render, or Heroku)
